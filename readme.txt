Creates a slider of the books of any norwegian author from bokhylla.no

== Description ==

Will create a slider of the books of any norwegian author using data from bokhylla.no using the shortocde

[bookslider firstname=xxx lastname=yyy]

== Installation ==

Upload this plugin to your blog, add the shortcode somewhere
